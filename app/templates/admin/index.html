Instructions:

Here Admin can able to see all the user details and edit,delete the details

The Process of Validating the Exam Registration Form submitted by user :

1) Go to User tab and check details of a particular user

2) Then Go to ExamRegistration tab and check details of a particular user and copy the "Aadhar card" document name and "Documents" name

3) Then go to user_documents > then look for copied file of a particular user

4) Then Open it and check it properly with the user details

5) If All details and documents are correct then go to user tab again

6) Then click on edit icon of a particular user and in " e_d " field enter "Yes" and save it

7) If user submitted documents are not valid along with it's details then dont do anything
