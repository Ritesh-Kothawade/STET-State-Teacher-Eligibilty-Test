Project Installation Steps-

Prerequisites-

- Install pycharm IDE Community version - https://www.jetbrains.com/pycharm/download/#section=windows

Installation Steps-

1. Open the "Maharashtra State Teachers Eligibility Test" folder in pycharm.

2. In Terminal enter command- flask run

3. The project will be served on -  http://127.0.0.1:5000/
